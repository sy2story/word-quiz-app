プライバシーポリシー

最終更新: 2026-05-20

「英単語クイズ」（以下「本アプリ」）は、Google スプレッドシートを単語帳として使う 学習用 Web アプリケーションです。本アプリの提供にあたって、運営者がユーザーから 取得・保管する情報はありません。本ポリシーは、その仕組みと前提を明示するものです。

1. 取得・保管する個人情報

運営者側のサーバーは存在しません。本アプリは静的な HTML / JavaScript として配信され、 すべての通信はユーザーのブラウザと Google API の間で直接行われます。 運営者がユーザーの個人情報・学習履歴・スプレッドシートの内容を 収集・閲覧・保管することは 一切ありません。

2. 使用する Google アカウントの権限

本アプリは、Google が提供する OAuth 認可の仕組みに基づき、以下のスコープを要求します。

https://www.googleapis.com/auth/drive.file

このスコープは 「アプリで開いたファイル、またはアプリで作成したファイルのみ」 にアクセスを許可するものです。具体的には、ユーザーが Google ピッカー画面で 自ら選択した1つのスプレッドシートだけが本アプリの読み書き対象となります。 ユーザーが選択していない他の Google ドライブのファイルには、本アプリは 一切アクセスできません。

3. スプレッドシートに書き込む情報

ユーザーが選択したスプレッドシートに対し、本アプリは以下の列を更新します。

  • correct_count / wrong_count: 正答数・誤答数
  • last_result / last_answered_at: 最終結果と回答時刻
  • is_weak / consecutive_correct_count: 苦手判定と連続正解数

これらはすべてユーザー自身の Google アカウント配下のスプレッドシートに直接書き込まれ、 運営者やその他の第三者が参照することはできません。

4. ブラウザに保存する情報（localStorage）

本アプリは、利便性のために以下の情報をユーザーの端末（ブラウザの localStorage）に保存します。 これらは端末内に保存されるのみで、外部に送信されることはありません。

  • 接続中スプレッドシートの ID / 名前 / URL
  • 苦手単語の ID 一覧、着手済みセクションの記録
  • 直近の単語データのキャッシュ（オフライン表示用）
  • 送信に失敗した解答結果の再送キュー

ブラウザの設定でサイトデータを削除すると、これらの情報も削除されます。

5. 第三者提供・解析ツール

本アプリは、アクセス解析・広告配信・行動トラッキングなどの第三者サービスを 一切使用していません。Cookie も使用していません。

6. アクセス権の取り消し

本アプリへのアクセス権限は、ユーザーがいつでも取り消すことができます。 Google アカウントの「サードパーティアプリとサービス」設定 から本アプリを削除すると、以降のアクセスは即座に無効化されます。

7. お問い合わせ

本ポリシーや本アプリに関するお問い合わせは、以下の GitHub Issues よりお願いします。

https://github.com/sy2story/word-quiz-app/issues

8. 改定について

本ポリシーの内容は予告なく変更されることがあります。重要な変更がある場合は、 本ページ上で通知します。
